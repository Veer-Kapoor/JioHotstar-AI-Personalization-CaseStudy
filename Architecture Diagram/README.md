# Architecture Diagram

This folder contains the **system architecture diagram** for the AI Personalization Engine designed for JioHotstar.

The diagram illustrates how the platform's AI recommendation system processes user signals, generates personalized recommendations, and continuously improves through feedback.

## Figma Diagram

View the full architecture diagram here:

(https://www.figma.com/make/B5dOfYynpTQRkQ4S7OXHH4/Personalization-Architecture-Diagram?t=4GchhR1Y1JB0Fi4x-1)

## What the Architecture Shows

- User platforms (Mobile, Web, Smart TV)
- API and personalization services
- Recommendation and ranking pipeline
- Machine learning models used for personalization
- Data pipelines and feature store
- Feedback loop for model improvement

This architecture demonstrates how the **AI-driven recommendation system scales to serve millions of users while delivering personalized content experiences**.
