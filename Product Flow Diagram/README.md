# Product Flow Diagram

This folder contains the **AI-powered content discovery product flow** for the JioHotstar personalization system.

The diagram illustrates how **user actions interact with the AI recommendation engine** to deliver personalized content experiences.

## Figma Diagram

View the full product flow diagram here:

https://www.figma.com/make/48Q7fYMPngPgt1mvi2vEH7/AI-Content-Discovery-Flow?t=YYtnT9BxoKGWnXH1-1

## What the Diagram Shows

- User journey from opening the app to watching content  
- AI recommendation pipeline triggered by user actions  
- Context detection (device, time, language)  
- Candidate generation and ranking models  
- Feedback signals used to improve recommendations  

This flow demonstrates how the **AI personalization engine continuously learns from user behavior to improve content discovery and engagement**.
