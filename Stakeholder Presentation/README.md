# Stakeholder Presentation

This folder contains the **stakeholder presentation** for the JioHotstar AI Personalization case study.

## File

**JioHotstar.pdf**

The presentation summarizes the proposed **AI-powered personalization engine**, highlighting the key problem, product strategy, solution design, and expected business impact.

## Purpose

This deck is designed for **product leaders and stakeholders** to quickly understand:

- The content discovery problem on JioHotstar  
- The AI-driven solution and product vision  
- Key features and system approach  
- Expected impact on engagement and retention  

It provides a concise overview of the full case study documented in the **PRD and supporting artifacts**.
