# User Journey Story

This folder contains the **user journey narrative** illustrating how AI personalization improves the streaming experience on JioHotstar.

## File

**Priyas-Morning-Commute.pdf**

The story follows **Priya**, a regional language viewer, and demonstrates how an AI-powered recommendation system reduces content discovery time and delivers more relevant recommendations.

## Purpose

This artifact helps visualize:

- Real user pain points in content discovery  
- How AI personalization improves the viewing experience  
- The impact of contextual and language-aware recommendations  

It complements the **PRD, system architecture, and product flow diagrams** in the overall AI Product Management case study.
