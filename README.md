# Overview

This repository contains a complete **AI Product Management case study** that proposes an AI-powered personalization engine for JioHotstar.

The goal is to improve **content discovery, engagement, and retention** on India's largest OTT platform.

---

# Problem

Streaming platforms face significant discovery challenges.

Key issues include:

• 100,000+ hours of content creating decision paralysis  
• Language mismatch for regional audiences  
• Poor recommendations for new users  
• Shared family profiles causing chaotic suggestions  

As a result, **28% of sessions end without any content being played.**

---

# Solution

The proposed **JioHotstar Personalization Engine (JPE)** uses AI to dynamically generate personalized content feeds.

Key capabilities:

• Context-aware recommendations  
• Language-first personalization  
• Explainable recommendations  
• Post-session content suggestions
